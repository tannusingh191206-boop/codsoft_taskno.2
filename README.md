# 🎓 Student Grade & CGPA Calculator

A simple Java Swing desktop application that calculates a student's:

- Total Marks
- Percentage
- CGPA
- Grade
- Pass/Fail Status

This project provides a user-friendly graphical interface for entering student details and subject marks and instantly generating results.

---

## 📌 Features

✅ Student Information Input

- Student Name
- Roll Number

✅ Subject Marks Entry

- Java
- DBMS
- Python
- Operating System
- Computer Network

✅ Automatic Calculation

- Total Marks
- Percentage
- CGPA

✅ Grade Generation

| Percentage | Grade |
|------------|--------|
| 90+ | A+ |
| 75 - 89 | A |
| 60 - 74 | B |
| 40 - 59 | C |
| Below 40 | F |

✅ Result Status

- PASS 🟢
- FAIL 🔴

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/student-grade-calculator.git
```

### 2. Navigate to Project Folder

```bash
cd student-grade-calculator
```

### 3. Compile the Program

```bash
javac StudentCalculatorUI.java
```

### 4. Run the Program

```bash
java StudentCalculatorUI
```

---

## 📊 Calculation Formula

### Percentage

```text
Percentage = Total Marks / 5
```

### CGPA

```text
CGPA = Percentage / 9.5
```

---

## 📷 Application Workflow

1. Enter Student Name
2. Enter Roll Number
3. Enter marks for all subjects
4. Click **Calculate Result**
5. View:
   - Total Marks
   - Percentage
   - CGPA
   - Grade
   - Pass/Fail Status
